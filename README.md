# erc20-mass-transfer
Usage: 
```
For main purpose:
1) 
npm install
2) 
./node_modules/.bin/truffle exec './src/erc20-mass-transfer.js' 
--network rinkeby --data './resources/erc20-data.csv' --batch 1
--token 29f1e20e825d88dc2f4aed65d1d8925cd71558ff

For test purpose:
1) 
npm install
2) 
./node_modules/.bin/truffle exec './src/erc20-mass-transfer.js' 
--network test --data './resources/erc20-data.csv' --batch 1 
--source 29f1e20e825d88dc2f4aed65d1d8925cd71558ff --token 29f1e20e825d88dc2f4aed65d1d8925cd71558ff
```
